### HTML 시맨틱 태그 사용 이유
- 시맨틱 태그란?
    - 웹 페이지 태그에 의미를 부여하는 태그 (의미를 가지는 태그)
        - 기존에는 div, span 등 의미 없는 블록 요소 사용
        - div(블록 형식 분할), span(인라인 형식 분할) 대신 의미 가지는 시맨틱 태그 권장됨
        - 유저 친화적 태그 통해 가독성 높임.
    ```
    header :머리말 (페이지 제목, 소개)
    nav : 하이퍼링크 모아둔 내비게이션
    aside : 본문 흐름에서 벗어난 노트 및 팁
    section : 문서의 장이나 절
    article : 본문과 독립적인 콘텐츠
    footer : 꼬리말 (저자, 저작권 등)
    ```
